About Pxe Cloud

Pxe cloud is a server where you can store all your images of systems in the cloud, ready to install using ipxe.

In Pxe Cloud we use the docker technology to start the services, and thus offer an easy installation.

Our services interact between: ipxe, dhcp, tftp, rethink, https, python3.6 and others.
